# Personal-Finance-Tracker
A simple and user-friendly application to track daily income and expenses, helping users manage their finances efficiently and stay within budget.

Personal Finance Tracker is a simple application designed to help users record and manage their daily income and expenses. It provides an easy way to monitor spending habits and improve financial planning.

---

🚀 Features

- Add income and expense entries
- Track daily transactions
- View total balance
- Simple and clean interface

---

🛠️ Technologies Used

- HTML
- CSS
- JavaScript

---

📖 How to Use

1. Open the project files
2. Run the "index.html" file in your browser
3. Add your income and expenses
4. Track your balance easily

---

🎯 Purpose

This project is created as a beginner-level project to learn and practice web development and basic financial tracking concepts.
